# Selene-Shop
Selene Shop is a PHP/MySQL e-commerce website for accessories. It includes user signup/login, product browsing from a database, cart and checkout, payment flow, contact page, and an admin panel for managing products. It uses sessions for auth/cart and cookies for past purchases.
